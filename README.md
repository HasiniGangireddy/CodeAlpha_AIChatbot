# Artificial Intelligence NLP Chatbot

This project is a Java-based Artificial Intelligence Chatbot developed using Java Swing and NLP concepts.

## Features

- GUI-based chatbot
- Natural Language Processing (NLP)
- Keyword matching
- Rule-based responses
- Interactive communication
- Java OOP concepts

## Technologies Used

- NLP Basics
- OOP Concepts
